Requisitos previos

  • Java 8 o superior instalado en tu sistema.
  • Conexión a Internet para acceder a la API de Star Wars.

Uso

Para ejecutar el programa, simplemente compila y ejecuta la clase Principal desde tu IDE o desde la línea de comandos. A continuación, sigue las instrucciones en pantalla para seleccionar el número de la película que deseas consultar. El programa guardará la información de la película en un archivo JSON en el directorio actual.

Ejemplo de Uso

java Principal

Después de ejecutar el comando anterior, sigue las instrucciones en pantalla para ingresar el número de la película que deseas consultar.
